# Project Management Tool

A full-stack MERN Project Management Application built using React, Node.js, Express.js, and MongoDB.

## Features

* User Registration and Login
* JWT Authentication
* Create Projects
* Edit Projects
* Delete Projects
* Create Tasks
* Update Task Status (Todo, In Progress, Done)
* Delete Tasks
* MongoDB Atlas Integration

## Tech Stack

### Frontend

* React
* Vite
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
